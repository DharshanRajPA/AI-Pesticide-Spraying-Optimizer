## 🚀 How to Run (3 Commands)

### **1. Install Dependencies**
```bash
pip install -r requirements.txt
```

### **2. Start Application**
```bash
python start.py
```
*OR double-click `start.bat` on Windows*

### **3. Open Browser**
Go to: **http://localhost:8000**
